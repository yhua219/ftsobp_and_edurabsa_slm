# The ACOS Laptop Datasets

All the ACOS Laptop datasets used by this study are direct adoption with de-tokenization, formatting change, and prompt addition to the original ACOS Laptop dataset by **Cai et al. (2021)**.

Full details about the original dataset are available at https://github.com/NUSTM/ACOS.


# The ASQP Rest16 Datasets

All the ASQP Rest16 datasets used by this study are direct adoption with de-tokenization, formatting change, and prompt addition to the original ASQP Rest16 dataset by **Zhang et al. (2021)**.

Full details about the original dataset are available at https://github.com/IsakZhang/ABSA-QUAD.
