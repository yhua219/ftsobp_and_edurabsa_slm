# Model Info - Phi4-mini-instruct

This study used the pre-trained Phi4-mini-instruct model obtained via https://huggingface.co/microsoft/Phi-4-mini-instruct ("the original model" hereafter). 

The original model was used for inference and training LoRA adaptors. No change was made to the original model.

The original model's licence and copyright notice files are included in this repository.
