# Model Info - Qwen-2.5-1.5B-Instruct

This study used the pre-trained Qwen-2.5-1.5B-Instruct model obtained via https://huggingface.co/Qwen/Qwen2.5-1.5B-Instruct ("the original model" hereafter). 

The original model was used for inference and training LoRA adaptors. No change was made to the original model.

The original model's licence (and copyright notice files if provided in the original model's repository) are included in this repository.
